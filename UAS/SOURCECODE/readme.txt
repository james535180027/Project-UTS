Yang perlu diperhatikan:

- Untuk mengakses halaman admin, silahkan klik tombol login pada navbar.
Kemudian masukkan "admin" untuk username dan password (tanpa tanda " ").

- Selain login sebagai admin, pengunjung nantinya juga dapat melakukan
login setelah melakukan registrasi. Untuk sekarang, silahkan masukkan
apapun selain "admin" pada username dan password. 
Misal: "user" untuk username dan "123456" untuk password (tanpa tanda " ").
